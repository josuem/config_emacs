* Comandos de Git y Github

** Comandos locales
| Comando                                     | Comentario                                                                              |
| git init                                    | Crea la carpeta .git que realiza la manteción del proyecto                              |
| git status                                  | Muestra los archivos en los cuales se está trabajando en el proyecto                    |
| git add                                     | Agrega un archivo para realizar su seguimiento. Con . o * agrega todos los archivos     |
| git commit                                  | Permite crear un mensaje de comentario para la versión respaldada                       |
| git commit -m "MENSAJE"                     | Agrega el comentario directamente, sin abrir el editor                                  |
| git config --global user.email "EMAIL"      | Configura el email de GitHub en el cliente local                                        |
| git config --global user.name "USUARIO"     | Configura el usuario de GitHub en el cliente local                                      |
| git log                                     | Muestra el log de los commit creados                                                    |
| git checkout -- "NOMBRE_ARCHIVO"            | Reestablece el archivo creado a su estatus guardado anteriormente                       |
| git diff "NOMBRE_ARCHIVO"                   | Muestra los cambios realizados del archivo                                              |
| git branch                                  | Muestra los hilos creados hasta el momento                                              |
| git branch "NOMBRE"                         | Crea un nuevo hilo para la modificación de los archivos                                 |
| git checkout "NOMBRE_BRANCH"                | Cambia el hilo de edición actual (entre branch)                                         |
| git commit --amend                          | Edita el último commit guardado, útil cuando falta agregar un archivo o cambios mínimos |
|                                             |                                                                                         |
| git config --global core.editor             | Cambia el editor utilizado por git a notepad++                                          |
| "'C:/Program Files/Notepad++/notepad++.exe' |                                                                                         |
| -multiInst -notabbar -nosession -noPlugin"  |                                                                                         |
